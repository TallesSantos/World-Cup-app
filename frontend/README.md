# React Native + Expo Go

Aplicação mobile desenvolvida com React Native utilizando Expo.

---

# Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- Expo Router

---

# Pré-requisitos

Antes de iniciar, instale:

- Node.js
- npm ou yarn
- Expo Go no celular

Links úteis:

- https://nodejs.org
- https://play.google.com/store/apps/details?id=host.exp.exponent

---

# Clonando o projeto

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git

```
# Baixando dependencias e subindo o projeto
  - baixar dependencias
```bash 
npm install
```
  - subindo o projeto
```bash
npx expo start
```
ou
```bash
npm run start
```
